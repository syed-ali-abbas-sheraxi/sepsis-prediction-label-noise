# Sepsis Prediction Under Label Noise

## Overview

An end-to-end machine learning pipeline for early sepsis onset prediction using the PhysioNet/CinC Challenge 2019 dataset.

The project addresses irregular ICU time-series, missing values, noisy physician labels, and temporal data leakage using robust preprocessing and XGBoost.

## Features

- Missing Value Imputation
- Label Noise Mitigation
- Temporal Leakage Prevention
- XGBoost Classification
- Threshold Optimization
- Model Calibration

## Dataset

PhysioNet/CinC Challenge 2019
